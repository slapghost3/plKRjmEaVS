# 项目简介

## 核心功能
- 登录
- 注册
- 图书信息管理
- 图书类型管理
- 借阅信息管理
- 用户管理
- 修改密码
- 模糊查询
- 分页查询
- 批量操作
- 图书借阅
- 图书归还

## 适用场景
- 图书馆信息管理
- 学校图书馆系统
- 企业图书馆管理
- 个人图书管理

## 技术特点
- 使用Java开发，采用SpringBoot框架
- 后端使用MySQL数据库
- 前端使用Vue框架
- 提供源码、文档及运行视频教程
- 支持运行环境配置与调试
